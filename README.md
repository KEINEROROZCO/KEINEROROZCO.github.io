## ✨ Sobre el Proyecto

Portafolio web personal construido desde cero con HTML, CSS y JavaScript vanilla. Diseño premium dark-mode con animaciones avanzadas, efectos visuales cinematográficos y experiencia de usuario de nivel profesional.

### 🎯 Características Principales

| Feature | Tecnología |
|---------|-----------|
| **Diseño Premium Dark** | CSS custom properties, glassmorphism, gradientes |
| **Partículas Interactivas** | Canvas API con conexiones al cursor |
| **Cursor Personalizado** | Trail effect + glow + estados hover |
| **Animaciones de Entrada** | Intersection Observer + CSS transitions |
| **Efecto Scramble Text** | JavaScript text animation en hero |
| **Typing Code Editor** | Efecto de escritura en la sección "Sobre mí" |
| **3D Tilt Cards** | Perspectiva CSS + mousemove tracking |
| **Efecto Magnético** | Botones que siguen el cursor |
| **Scroll Progress** | Barra de progreso con gradiente |
| **Formulario de Contacto** | FormSubmit.co integration |
| **SEO Completo** | Open Graph, Twitter Cards, JSON-LD, sitemap |
| **Responsive Design** | Mobile-first, 4 breakpoints |
| **Accesibilidad** | Skip links, ARIA, contraste |

Plataforma de e-commerce de alta gama con productos generados dinámicamente (Faker.js), imágenes de Unsplash, carrito funcional con checkout y diseño tipo Apple Store.

### 2. Tienda de Barrio Online
